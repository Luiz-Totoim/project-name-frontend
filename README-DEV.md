# Desenvolvimento — Etapa 0 (esqueleto)

Este branch contém um esqueleto mínimo para iniciar o desenvolvimento com Create React App.

Como rodar:

1. Instalar dependências:

```powershell
npm install
```

2. Rodar em modo desenvolvimento:

```powershell
npm start
```

Observações:
- Este é um esqueleto inicial. Próximo passo: portar marcação para JSX e implementar componentes.
- O projeto usa `react-scripts` (CRA). Se preferir Vite, posso gerar esqueleto com Vite em outra branch.
